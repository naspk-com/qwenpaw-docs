QwenPaw 在飞牛上由两个部分组成：

- **飞牛控制台**：fnOS 侧边栏中的管理界面，提供服务状态监控、版本升级、日志查看、外网访问等功能
- **QwenPaw 本体**：运行在端口 19091 的 Web 应用，用于管理智能体、通道、技能等 AI 功能

> **提示**：本文档主要介绍 QwenPaw 在飞牛上的安装方式和飞牛控制台功能。关于 QwenPaw 本体的使用（如智能体创建、Skills 开发等），请访问 [QwenPaw 官网](https://qwenpaw.agentscope.io)。

## 飞牛控制台

![飞牛控制台](/images/poster-console.png)

安装完成后，在飞牛桌面点击 **QwenPaw** 图标即可进入飞牛控制台。飞牛控制台包含以下功能页面：

| 页面 | 说明 |
|------|------|
| **控制台首页** | 查看 QwenPaw 服务状态、版本号、运行时长；提供打开本体、重启、升级、重置密码等操作 |
| **运行日志** | 查看 QwenPaw 运行日志（最新 500 条），支持自动刷新、关键字筛选、分页浏览 |
| **外网访问** | 集成 Cloudflare Tunnel，一键注册域名实现外网访问 QwenPaw 控制台 |
| **QQ群交流** | 加入飞牛QwenPaw使用交流群，获取帮助 |
| **关于** | 作者信息、文档库链接、赞赏支持 |

详见 [控制台功能说明](/setup/dashboard)。

## QwenPaw 本体

点击飞牛控制台首页的「打开 QwenPaw」按钮，或在浏览器访问 `http://<NAS_IP>:19091`，即可进入 QwenPaw 本体控制台。

QwenPaw 本体由 [AgentScope](https://github.com/agentscope-ai) 团队构建，提供以下核心能力：

- **多通道对话**：支持微信、QQ、钉钉、飞书、Discord、Telegram 等十余种消息通道
- **多智能体协作**：独立配置多个 Agent，各自拥有独立的记忆和技能
- **定时任务**：设置 Cron 定时任务自动执行
- **Skills 扩展**：能力由 Skills 决定，支持无限扩展

本体功能的详细文档请访问 [QwenPaw 官网](https://qwenpaw.agentscope.io)。

## 前置要求

- 飞牛 NAS 系统 **>= 0.9.21**
- 如需使用内置 Ollama 驱动，请先在飞牛应用商店安装 **Ollama** 应用

## 安装

QwenPaw 已上架飞牛应用商店，可一键安装。详见 [安装指南](/getting-started/installation)。

## 依赖说明

安装包会自动处理以下依赖：

- **Python 3.12**：Python 运行环境
- **Node.js v24**：前端构建和运行时环境
- **Cloudflare Tunnel**（可选）：用于外网访问
