## 通过应用商店安装

QwenPaw 作为飞牛原生应用，可通过飞牛应用商店一键安装，无需手动下载或配置复杂依赖。

![飞牛应用商店安装](/images/installation.png)

### 安装步骤

1. 打开飞牛 Web 管理后台
2. 进入「应用中心」
3. 搜索「QwenPaw」
4. 点击「安装」
5. 在安装向导中，同意 「安装QwenPaw - 协议许可」![协议许可](/images/installation-agreement.png)
6. 选择PyPI安装源，默认选择「清华大学镜像源」 ![清华源](/images/installation-pypi.png)
7. 选择日志级别（默认 `info`，调试可选 `debug`）![日志级别](/images/installation-loglevel.png)
8. 点击「下一步」，等待安装完成![安装完成](/images/installation-complete.png)

> **注意**：安装过程中需要创建 Python 虚拟环境并从 PyPI 安装 QwenPaw 及其依赖，进度条在 45% 左右可能停留较长时间，属于正常现象，请耐心等待。

![安装进度](/images/installation-progress.png)

应用安装完成后，QwenPaw 会出现在飞牛侧边栏中。点击图标即可进入控制台管理界面。

## 依赖说明

QwenPaw 安装时会自动安装以下依赖：

| 依赖 | 说明 |
|------|------|
| Python 3.12 | Python 运行环境 |
| Node.js v24 | 前端构建和运行时环境 |
| Cloudflare Tunnel | 可选依赖，用于将控制台穿透到公网访问 |

> **提示**：如需使用 Ollama 本地模型，请在飞牛应用商店先安装 **Ollama** 应用。连接 Ollama 时 API Key 字段不能为空，可以填写任意非空字符串（如 `ollama`）。

## 首次启动

安装后首次打开 QwenPaw，你会看到控制台界面。你需要先完成以下初始配置：

1. **配置模型**：在「模型」页面添加至少一个大语言模型（支持 OpenAI API、Ollama 本地模型等）
2. **创建智能体**：在「智能体」页面创建你的第一个 Agent
3. **开始对话**：在「对话」页面与智能体开始聊天

以上功能的详细说明请参考 [控制台功能](/setup/dashboard)。

## 远程访问

如果你想在外网访问 QwenPaw 控制台，推荐使用飞牛应用商店中的 [Cloudflare Tunnel](https://tunnel.naspk.com) 应用。通过 Tunnel 创建一条安全隧道，无需公网 IP 即可通过域名访问。

## 更新应用

当飞牛应用商店提示有新版本时，点击更新即可。更新过程中可选择保留现有运行环境以加快升级速度。更新不会丢失已有的配置和数据。

## 卸载应用

如需卸载，在飞牛应用中心找到 QwenPaw，点击卸载。卸载前建议：

1. 创建一次备份，便于以后恢复
2. 确认不再需要的数据可以删除
