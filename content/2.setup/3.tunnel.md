「外网访问」页面集成了 [Cloudflare Tunnel](https://tunnel.naspk.com) 应用的能力，让你无需公网 IP、无需配置路由器端口转发，即可通过自定义域名从外网访问 QwenPaw 控制台。

![外网访问页面](/images/console-tunnel.png)

## 前置条件

在使用外网访问功能前，请确保：

1. 已在飞牛应用商店安装 **Cloudflare Tunnel** 应用
2. 已在 Tunnel 应用中完成 [Cloudflare 账号配置](https://tunnel.naspk.com/configuration/cloudflare-credentials)
3. 拥有一个已接入 Cloudflare 的域名
4. 飞牛QwenPaw版本 >= 1.1.11

上述条件不满足时，页面会提示无法获取 Tunnel 状态。

## 页面组成

页面分为两个功能卡片：

### Cloudflare Tunnel 状态

展示 Tunnel 服务的整体运行情况，帮助你确认外网访问通道是否就绪。

| 信息 | 说明 |
|------|------|
| Tunnel ID | 当前连接的 Tunnel 标识（显示前 16 位） |
| 架构 | 设备 CPU 架构（如 amd64 / arm64） |
| 进程 PID | cloudflared 进程的 PID |
| 健康状态 | healthy（正常）/ 异常 |

状态徽章颜色含义：

- **绿色「运行中」**：Tunnel 已连接 Cloudflare，通道正常
- **黄色「异常」**：Tunnel 已连接但健康检查未通过
- **灰色「未运行」**：Tunnel 服务未启动

### 域名注册状态

展示当前 QwenPaw 的外网域名注册情况。

**未注册状态**：显示域名注册表单，等待你填写信息。

**已注册状态**：展示以下详细信息：

| 信息 | 说明 |
|------|------|
| 注册域名 | 已绑定的公网域名 |
| 本地服务 | 域名指向的内网服务地址 |
| DNS 记录 | 域名解析是否已生效 |
| Ingress 规则 | Tunnel 转发规则是否已配置 |
| Tunnel 状态 | Cloudflare Tunnel 是否在运行 |

当 DNS 记录和 Ingress 规则都生效时，状态显示为绿色「已生效」，此时即可通过域名访问 QwenPaw。

## 注册外网域名

### 第一步：填写域名

在「域名」输入框中填写你希望使用的公网域名，例如：

- `qwenpaw.yourdomain.com`
- `ai.yourdomain.com`

> **注意**：域名必须已接入 Cloudflare，否则 DNS 记录无法自动创建。如果你还没有域名，可以前往 Cloudflare 注册。

### 第二步：确认服务地址

「本地服务地址」默认为当前 NAS 上的 QwenPaw 本体地址（`http://<NAS_IP>:19091`），通常无需修改。

如果你修改了 QwenPaw 本体的端口，请同步更新此处的服务地址。

> **注意**：本地服务地址请使用 `http://` 协议，不要使用 `https://`。

### 第三步：点击注册

点击「注册域名」按钮，系统会自动完成以下操作：

1. 在 Cloudflare 创建 DNS CNAME 记录，将域名指向 Tunnel
2. 配置 Tunnel 的 Ingress 规则，将域名流量转发到本地服务
3. 如果 Tunnel 未连接，自动启动 Tunnel 服务

### 第四步：等待生效

注册成功后，DNS 记录需要几分钟时间在全球生效。生效后，状态卡片会变为绿色「已生效」。

此时在浏览器访问你的域名（如 `https://qwenpaw.yourdomain.com`），即可在外网打开 QwenPaw 控制台。

## 常见问题

### 注册失败怎么办？

1. 确认域名已接入 Cloudflare（域名 DNS 服务器指向 Cloudflare）
2. 确认 Tunnel 应用中的 API Token 具有 DNS 读写权限
3. 确认 Tunnel 应用已完成账号配置
4. 查看 Tunnel 应用的 [运行日志](https://tunnel.naspk.com/advanced/logs)

### 注册成功但域名无法访问？

1. DNS 记录需要时间生效，通常等待 2-5 分钟
2. 检查 QwenPaw 服务是否正常运行（查看控制台首页状态）
3. 访问时请使用 `https://` 协议（Cloudflare Tunnel 默认启用 HTTPS）

### 如何取消外网访问？

如需取消外网访问，请在 [Cloudflare Tunnel](https://tunnel.naspk.com) 应用中删除对应的域名规则或断开 Tunnel 连接。QwenPaw 本地服务不受影响。
