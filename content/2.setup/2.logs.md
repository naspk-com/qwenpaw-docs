「运行日志」页面用于查看 QwenPaw 服务运行过程中产生的日志，是排查问题的主要工具。日志最多保留最新 500 条，包含服务运行日志和升级日志。

![运行日志页面](/images/console-logs.png)

## 日志条目

每条日志包含以下信息：

| 字段 | 说明 |
|------|------|
| 日志级别 | `info` / `debug` / `warning` / `error` |
| 时间 | 日志产生的时间戳 |
| 内容 | 日志的详细消息文本 |

### 日志级别颜色

不同级别的日志以不同颜色标识，便于快速定位：

| 级别 | 颜色 | 含义 |
|------|------|------|
| info | 绿色 | 一般信息，如服务启动、配置加载 |
| warning | 橙色 | 警告信息，可能影响但不阻断功能 |
| error | 红色 | 错误信息，需要关注和处理 |
| debug | 灰色 | 调试信息，通常无需关注 |

## 功能操作

### 自动刷新

日志页面默认开启自动刷新，每 3 秒自动拉取最新日志。你可以通过顶部的开关控制是否自动刷新。

**使用建议**：
- 排查实时问题时保持自动刷新开启
- 阅读历史日志时关闭自动刷新，避免页面跳动

### 关键字筛选

在搜索框中输入关键字，可以快速过滤包含指定内容的日志条目。支持中英文关键字。

### 分页浏览

日志支持分页显示，可选择每页显示的条数：

| 选项 | 说明 |
|------|------|
| 10 条/页 | 默认，适合快速浏览 |
| 20 条/页 | 适合一般查阅 |
| 50 条/页 | 适合批量查看 |
| 100 条/页 | 适合导出或密集排查 |

### 刷新与清空

- **刷新按钮**：手动拉取最新日志，并弹出刷新成功提示
- **清空按钮**：清除当前内存中的日志缓存

## 排障流程

推荐使用以下流程进行问题排查：

1. 点击「清空」按钮，清除旧日志
2. 回到控制台首页，执行可能出问题的操作（如重启、升级）
3. 返回日志页面，观察新产生的日志
4. 使用关键字筛选定位错误信息
5. 根据错误内容搜索解决方案，或加入 [QQ 群](https://qm.qq.com/q/uydPbFJ4ic) 求助
