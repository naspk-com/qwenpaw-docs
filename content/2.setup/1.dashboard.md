控制台首页是你打开 QwenPaw 后看到的第一个页面，展示 QwenPaw 服务的运行状态和版本信息，并提供常用的服务管理操作入口。

![控制台首页概览](/images/console-dashboard.png)

## 服务状态

页面顶部以卡片形式展示 QwenPaw 服务的实时状态信息：

| 信息 | 说明 |
|------|------|
| 服务状态 | 显示「运行中」或「已停止」，绿色表示正常运行 |
| 版本号 | 当前安装的 QwenPaw 版本（如 v1.1.11） |
| 运行时长 | 服务启动后已持续运行的时间（格式：`天 时:分:秒`） |
| 认证状态 | 是否启用了 QwenPaw 本体的访问密码保护 |

服务状态信息通过后台 API 实时获取，页面会持续刷新运行时长。如果服务意外停止，你可以在此页面快速发现问题。

## 常用操作

首页提供四个核心操作按钮：

### 打开 QwenPaw

点击「打开 QwenPaw」按钮，会在新标签页中打开 QwenPaw 本体控制台（`http://<NAS_IP>:19091`）。这是你管理智能体、配置消息通道、安装技能等操作的主界面。

> **提示**：QwenPaw 本体的详细使用说明，请访问 [QwenPaw 官网](https://qwenpaw.agentscope.io)。

### 重启服务

当 QwenPaw 服务出现异常（如响应缓慢、连接断开等）时，可以点击「重启服务」进行恢复。

操作流程：
1. 点击「重启服务」按钮
2. 确认重启操作
3. 等待服务重新启动（通常需要 10-30 秒）
4. 页面自动刷新状态

重启后建议等待几秒，确认服务状态恢复为「运行中」后再进行其他操作。

### 重置密码

如果你忘记了 QwenPaw 本体的访问密码，可以点击「重置密码」清除密码保护。

操作流程：
1. 点击「重置密码」按钮
2. 在确认弹窗中确认操作
3. 密码清除后，QwenPaw 本体将不再要求登录认证

![重置密码确认](/images/console-reset-auth.png)

> **注意**：重置后任何人都可以访问 QwenPaw 本体控制台，建议重置后尽快设置新密码。

### 版本升级

点击「版本升级」可将 QwenPaw 升级到最新版本。

#### 升级流程

升级操作分为以下步骤：

**第一步：确认升级**

点击「版本升级」按钮后，系统会弹出免责声明提示。升级操作会影响当前正在运行的服务，请确保没有重要任务正在执行。

![升级免责声明](/images/console-upgrade-disclaimer.png)

**第二步：数据备份（可选但建议）**

系统会提示你先备份数据，以防升级过程中出现意外。

- 点击「立即备份并下载」：将 QwenPaw 配置数据打包下载到本地
- 点击「跳过备份」：不备份直接进入下一步

![升级备份提示](/images/console-upgrade-backup.png)

**第三步：选择镜像源**

选择一个 PyPI 镜像源用于下载更新包：

| 镜像源 | 适用场景 |
|--------|------|
| PyPI 官方源 | 海外网络环境或不需要加速 |
| 清华大学镜像源 | 国内用户推荐 |
| 阿里云镜像源 | 国内用户备选 |
| 中科大镜像源 | 国内用户备选 |

![选择镜像源](/images/console-upgrade-mirror.png)

**第四步：等待升级完成**

选择镜像源后点击「确认升级」，系统开始下载并安装更新。升级过程中会实时显示日志输出：

![升级进度](/images/console-upgrade-progress.png)

升级完成后页面会显示结果提示。如果升级失败，请查看 [运行日志](/setup/logs) 排查具体原因。
